# Temple Accounting Software

## Overview

Temple Accounting Software is a desktop application developed using C#.NET Windows Forms and MS Access.

The software was developed to manage daily accounting operations for a temple office, including ledger management, transaction tracking, account statements, and balance calculations.

The application is currently being used for office operations.

## Features

* User Login
* Ledger Creation
* Account Management
* Credit and Debit Entry
* Opening Balance Management
* Account Statement Generation
* Transaction History
* Financial Reports

## Technologies Used

* C#.NET
* Windows Forms
* MS Access Database
* OLEDB Connectivity
* Visual Studio

## Project Structure

* Login Module

* Transaction Management
